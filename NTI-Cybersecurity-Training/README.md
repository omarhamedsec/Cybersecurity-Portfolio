# 🎓 NTI Cybersecurity Training

This directory documents the practical labs and hands-on exercises completed during the NTI (National Telecommunication Institute) Cybersecurity Training.

## 📂 Contents

### 1. `Labs-and-Writeups/`
This folder is the core of this training, featuring practical, scenario-based labs:
- **Network Analysis**: `Lab 5.3.7 intro to wireshark`, `Lab 10.6.7 http & https`, `Lab 10.2.7 wire udp dns`
- **Reconnaissance & Scanning**: `Lab 9.3.8 nmap`
- **Offensive Security**: `Lab 17.2.6 attacking a mysql database`
- **System Administration**: `Lab 3.3.11 powershell`, `Lab 4.2.7 linux shell`
- **Cryptography**: `Lab 21.2.11 encrypting and decrypting`

### 2. `Summaries-and-Notes/`
Contains theoretical summaries ("زتونه") and module overviews to solidify the concepts learned.

### 3. `LinkedIn-Writeups/`
Videos and write-ups created to share knowledge and practical application, including my "Record lab 5.3.7 intro to wireshark".

### 4. `APPS/` & `LABS/`
Additional applications and raw lab files utilized during the training.

## 💡 Highlighted Lab: Wireshark Analysis
Analyzing network traffic is a crucial skill for any SOC analyst. In these labs, I utilized Wireshark to:
- Capture and filter TCP/UDP packets.
- Analyze unencrypted HTTP traffic vs encrypted HTTPS traffic.
- Investigate DNS queries and responses.
